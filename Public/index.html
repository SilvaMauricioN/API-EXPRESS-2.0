Colección

Este endpoint da acceso a la colección con una breve información sobre cada objeto. Los resultados se dividen en páginas. Utilizando los parámetros `pagina` y `limite` puede obtener más resultados, hasta un total de 10.000.

URL Base de Colección

http://localhost:8080/api/museorijks/coleccion

Parámetros

Se admiten los siguientes parámetros:

Parámetro Formato Por defecto Descripción
id a-z 0-9 - Obligatorio: agregue su clave API a cada solicitud.
pagina 0-n 1 La página de resultados.
limite 1-30 20 El numero de resultados por página.
autor a-z - Este agente debe crear el objeto. Distingue entre mayúsculas y minúsculas. Por ejemplo:Rembrandt+van+Rijn

Nota: Ten en cuenta que `pagina * limite` no debe superar 10.000.

http://localhost:8080/api/museorijks/coleccion

Detalles de la colección

Este endpoint proporciona más detalles sobre un objeto específico. Los números de objeto se pueden obtener a partir de los resultados proporcionados en el endpoint Colección.

Ejemplo

http://localhost:8080/api/museorijks/coleccion/obra/{object-number}

Parámetros

Se admiten los siguientes parámetros:

Parámetro Formato Por defecto Descripción
id a-z 0-9 - Obligatorio: agregue su clave API a cada solicitud.
object-number a-z 0-9 - El identificador del objeto (distingue entre mayúsculas y minúsculas).
