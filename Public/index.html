Introducción y Guía de Uso

Esta API permite la consulta de obras de la colección del Museo Rijks. Está diseñada para ser simple y de solo lectura (GET).

Cómo Empezar:

  1. Obtén tu Clave: Asegúrate de tener tu clave API. Esta debe ser enviada en el parámetro `key` en cada solicitud.
  2. Consulta la Colección: Usa el enlace "Colección" para listar obras y obtener sus identificadores.
  3. Consulta el Detalle: Usa el identificador obtenido para consultar los detalles de una obra específica.

Solicitar API Key

Completa el siguiente formulario para solicitar acceso a la API.

Guarde su API KEY

Guarde su clave secreta en un lugar seguro, ya que no podrá volver a verla. Manténgala a buen recaudo, ya que cualquier persona que tenga su clave API puede realizar solicitudes en su nombre. Si la pierde, tendrá que generar una nueva.

Correo electrónico ya registrado. Esta dirección de correo electrónico ya está asociada a una clave API. Utilice otra dirección de correo electrónico.
