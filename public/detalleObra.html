Detalles De La Obra

Este endpoint proporciona más detalles sobre un objeto específico. Los números de objeto se pueden obtener a partir de los resultados proporcionados en el endpoint Colección.

MÉTODO: GET
Ejemplo

https://museo-dwgk.onrender.com/api/museorijks/obra{objectnumber}

Parámetros

Este endpoint requiere un parámetro de ruta y un parámetro de consulta obligatorio:

Parámetro Formato Por defecto Descripción
Key a-z 0-9 - Obligatorio: agregue su clave API a cada solicitud.
objectnumber a-z 0-9 - El identificador del objeto (distingue entre mayúsculas y minúsculas).

Ejemplo de Detalle de la Coleccion

Petición de Ejemplo

https://museo-dwgk.onrender.com/api/museorijks/obra/SK-A-3262
